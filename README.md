# Mini CRM System
## Overview

Mini CRM (Customer Relationship Management) is a web-based application designed to help businesses efficiently manage customer information, track leads, monitor interactions, and improve overall customer engagement.

## Features

* Customer Information Management
* Lead Tracking and Management
* Contact Details Storage
* Dashboard with Key Insights
* Search and Filter Functionality
* Responsive User Interface
* Secure User Authentication
* Data Management and Reporting

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Objectives

* Simplify customer data management.
* Improve lead tracking and follow-up processes.
* Provide an easy-to-use interface for managing customer relationships.
* Enhance productivity through organized customer records.

## Installation

1. Clone the repository:

   ```bash
   git clone <your-github-repository-link>
   ```

2. Open the project folder.

3. Install dependencies (if applicable):

   ```bash
   npm install
   ```

4. Run the project:

   ```bash
   npm start
   ```

## Usage

* Register/Login to the system.
* Add and manage customer records.
* Track leads and interactions.
* View customer details through the dashboard.
* Update and organize customer information efficiently.
